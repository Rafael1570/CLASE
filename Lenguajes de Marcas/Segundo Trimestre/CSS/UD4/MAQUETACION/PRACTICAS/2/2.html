HTML & CSS

Descripción de CSS

CSS representa las siglas de Cascading Style Sheets (hojas de estilo en cascada), que podemos definir como el lenguaje que nnos permite controlar el aspecto visual de nuestro documento HTML.


HTML y CSS se complementan para separar el contenido su presentación

Esta división nos aporta numerosas ventajas, puesto que, asl ser independientes el contenido y el aspecto visual, para un mismo contenido podremos tener diferentes visualizaciones. Esta característic es fundamental para poder trabajar en distintos dispositivos.
